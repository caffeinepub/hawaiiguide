# Specification

## Summary
**Goal:** Build a Hawaii travel guide app with island browsing, detail pages, and a lush tropical visual theme.

**Planned changes:**
- Backend stores structured data for each of the six main Hawaiian islands (Big Island, Maui, Oahu, Kauai, Molokai, Lanai), including name, description, top attractions, beaches, and travel tips, with queries to fetch all islands or a single island by ID
- Landing/home page with a full-bleed hero banner image evoking Hawaii
- Island browsing page with card-based layout showing each island's thumbnail image, name, and brief info
- Island detail page showing attractions, beaches, and travel tips for the selected island
- Navigation between home, island list, and individual island detail views
- Tropical visual theme: coral, turquoise, golden yellow, and deep green palette; bold serif headings; clean sans-serif body text; immersive section dividers
- Static images loaded from `frontend/public/assets/generated` for the hero banner and each island card

**User-visible outcome:** Users can explore a visually rich Hawaii travel guide, browse all six islands on card-based layouts, and dive into each island's highlights, beaches, and travel tips.
